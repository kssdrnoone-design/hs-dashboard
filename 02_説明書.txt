■ 高校受験ダッシュボード 使い方 （2026/05/04 大規模改修版）

詳細は README.md を参照。ここは要点のみ。

============================================================
【全体の流れ】
============================================================
  1. PCで 00_高校情報収集.bat をダブルクリック
        ↓
  2. 24校を自動巡回 → 最新HTML生成 → GitHub Pages に自動push
        ↓
  3. スマホアプリ起動 → 自動で最新表示
        ↓
  4. 予約は ✅予約済 タブから家族全員が追加・編集
     → GAS経由で全端末に自動同期

============================================================
【主要機能】
============================================================
  🆕 新着       : 前回スクレイプからの新規イベント
  ✅ 予約済     : 家族で共有する予約説明会（GAS同期）
  📅 カレンダー : 月別グリッド（予約は⭐金枠で強調）
  🏫 学校       : 24校を偏差値順カード表示
  📊 比較       : ソート可能な比較表
  📋 入試       : 都立入試スケジュール
  🗺 地図       : 国土地理院淡色地図 + 通学経路リンク

============================================================
【スマホアプリ最新版】
============================================================
ファイル: 高校受験_v1.2_20260504.apk
サイズ: 約 5.6MB

家族への配布:
  Googleドライブ/LINE等で送信 → スマホでタップしてインストール
  GAS URLはアプリに組み込み済み → 家族側の設定一切不要

公開URL（ブラウザ閲覧用）:
  https://kssdrnoone-design.github.io/hs-dashboard/

============================================================
【予約の追加・削除】
============================================================
  追加・編集・削除 → 自動でGAS経由で全端末に共有
  他端末で反映するには:
    - 「🔄 同期」を押す
    - またはアプリを起動し直す（自動で最新取得）

  ⚠️ オフライン追加した直後に「🔄 同期」を押すと
     リモートに無いデータは消える可能性。
     編集後はオンライン復帰を待ってから同期。

============================================================
【データ実体】
============================================================
  予約データ: Googleドライブ「hs-dashboard-reservations」
              スプレッドシートで人間可読
  GAS Webアプリ URL: 03_config.json の gas_url に記載
  GASスクリプト: gas_sync.gs (リポジトリに同梱)

============================================================
【APK再ビルドが必要な場合】
============================================================
  通常不要（GitHub Pages更新で全機能反映）

  必要な場合:
    - Android本体ロジック変更（SwipeRefresh等）
    - オフライン用内蔵HTMLを最新化したい時

  実行: 99_APK再ビルド.bat （ビルド約2-3分）

============================================================
【学校リストの編集】
============================================================
  03_config.json の schools 配列を編集 → bat再実行で反映
  偏差値は minkou.jp ソースで手動メンテ（年1回）
  中学校情報のフィルタは EXCLUDE_KEYWORDS で制御
