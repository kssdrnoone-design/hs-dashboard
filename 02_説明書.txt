■ 高校受験ダッシュボード 使い方

【初回セットアップ】
1. Python 3.13 と Playwright が入っていること（ニュース収集ツールと同じ環境）
   入ってなければコマンドプロンプトで:
     pip install playwright
     playwright install chromium

2. 04_.env に Gmail認証情報が入っていること（ニュース収集ツールからコピー済み）

【使い方】
1. 「00_高校情報収集.bat」をダブルクリック
2. 自動でブラウザが立ち上がり25校を巡回 → 説明会日程を抽出
3. 完了するとブラウザにダッシュボードが表示される
4. 新着イベント検知時は Gmail 送信先に通知メールが届く

【タブ構成】
- 🆕 新着       : 前回収集からの新しいイベントのみ
- 📅 カレンダー : 直近の説明会・オープンスクール一覧
- 🏫 学校カード : 対象25校を偏差値順に表示
- 📊 比較表     : カラムクリックでソート可能
- 📋 入試日程   : 都立の入試スケジュール（Phase2で自動化予定）

【学校リストの編集】
03_config.json を編集することで学校の追加・削除・偏差値更新が可能
偏差値は年1回手動更新してください

【データ保存場所】
- data/latest.json        : 最新スナップショット（差分検知用）
- data/raw_YYYYMMDD.json  : 日次アーカイブ
- reports/index.html      : 最新ダッシュボード
- reports/dashboard_YYYYMMDD.html : 日次アーカイブ

【注意】
- 各校サイトへの負荷軽減のため1校あたり2-3秒の待機を挟みます（全25校で約1分）
- URLが変更された学校は取得失敗として表示されます（config.json修正必要）
- 偏差値は公式発表が無いため手動入力方式です
