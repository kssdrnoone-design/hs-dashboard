■ 高校受験ダッシュボード 使い方 （Phase2+Phase3 完成版）

============================================================
【全体の流れ】
============================================================

  1. PCで 00_高校情報収集.bat をダブルクリック
        ↓
  2. 25校を自動巡回 → 最新ダッシュボードHTML生成
        ↓
  3. GitHub Pages に自動アップロード (git push)
        ↓
  4. 親と息子のスマホにインストール済みアプリから、
     最新データが自動で見られる！

※ メール通知は使わないのでセットアップ不要（.env不要）

============================================================
【スマホアプリ（APK）の配布】
============================================================

ファイル: 高校受験_v1.0.apk
サイズ: 約 5.7MB

親と息子のスマホにインストールする手順:
  1. APKファイルをGoogleドライブ/LINE/メール等で送る
  2. スマホで受け取ったら、ファイルをタップして開く
  3. 「提供元不明のアプリ」の警告が出たら「設定から許可」
  4. インストール完了
  5. ホーム画面の「高校受験」アイコンをタップ
  6. 自動でGitHub Pagesから最新データを表示

機能:
  - オンライン時: GitHub Pagesから最新データを取得
  - オフライン時: APK内蔵の固定版HTMLを表示
  - プルリフレッシュ: 画面を下に引っ張ると再取得
  - 学校公式リンクをタップ → 外部ブラウザで学校HP

公開URL: https://kssdrnoone-design.github.io/hs-dashboard/
（このURLを直接ブラウザで開いても同じダッシュボードが見られる）

============================================================
【毎日の運用】
============================================================

00_高校情報収集.bat をダブルクリックするだけ:
  - 25校巡回 (約1-2分)
  - GitHub Pages に自動デプロイ
  - スマホアプリ側は次回起動時に自動で最新化

タスクスケジューラ登録推奨: 毎朝7時頃に自動実行

============================================================
【APK再ビルドが必要なタイミング】
============================================================

通常は GitHub Pages 更新だけで最新情報がアプリに届くので、
APK再ビルドは不要。

以下の場合だけ 99_APK再ビルド.bat を実行:
  - オフライン表示用の内蔵HTMLを最新化したい時
  - アプリ本体のロジックを変えた時
  - 公開URLを変更した時

ビルド時間: 約2-3分

============================================================
【初回セットアップ】
============================================================

1. Python 3.13 と Playwright
   （ニュース収集ツールと同じ環境で動作）

2. Git と GitHub CLI (gh)
   認証: gh auth login で kssdrnoone-design として認証済み

3. Android ビルド環境 （99_APK再ビルド.bat用）
   - JAVA_HOME: C:\Program Files\Android\Android Studio1\jbr
   - ANDROID_HOME: C:\Users\kssdr\AppData\Local\Android\Sdk
   - Gradle 8.11.1 (.gradle配下)

============================================================
【タブ構成】
============================================================
  🆕 新着       : 前回収集からの新しいイベント
  📅 カレンダー : 直近の説明会・オープンスクール一覧
  🏫 学校カード : 対象25校を偏差値順に表示
  📊 比較表     : カラムクリックでソート可能
  📋 入試日程   : 都立の入試スケジュール

============================================================
【学校リストの編集】
============================================================

03_config.json を編集することで学校の追加・削除・偏差値更新が可能
偏差値は年1回手動更新してください

url_event フィールド: 学校の説明会ページURLを直接指定可能
（指定がなければトップページからサブページ追跡）

============================================================
【データ保存場所】
============================================================
  index.html              : GitHub Pages配信用（ルート）
  reports/index.html      : ローカル閲覧用
  reports/dashboard_*.html: 日次アーカイブ（Git除外）
  data/latest.json        : 差分検知用スナップショット（Git除外）
  data/raw_*.json         : 日次生データ（Git除外）

============================================================
【注意】
============================================================

- 1校あたり2-3秒の待機あり（全25校で約1-2分）
- URL変更された学校は取得失敗として表示
- 偏差値は公式発表が無いため手動入力方式
