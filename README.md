# 高校受験ダッシュボード

長男（中3・恋ヶ窪在住）の高校受験用。多摩エリア24校の説明会日程を自動巡回し、家族で予約管理＆共有するシステム。

- 公開URL: https://kssdrnoone-design.github.io/hs-dashboard/
- リポジトリ: https://github.com/kssdrnoone-design/hs-dashboard
- スマホアプリ最新版: `高校受験_v1.2_20260504.apk`

---

## 全体構成

```
[1] スクレイピング（PC・bat一発）
    24校の公式HP → 説明会日程抽出 → HTML生成
        ↓ git push
[2] GitHub Pages（公開URL）
    https://kssdrnoone-design.github.io/hs-dashboard/
        ↓
[3] スマホアプリ（WebViewラッパー APK）
    家族のスマホに配布。GitHub Pages を表示
        ↓
[4] 予約共有（GAS Webアプリ）
    Google Apps Script + スプレッドシート
    家族全員が予約を追加/編集/削除 → 自動同期
```

## 日常運用

### スクレイプ（毎週〜数週ごと）

1. PCで `00_高校情報収集.bat` をダブルクリック
2. 24校巡回（約1〜2分） → HTML生成 → GitHub Pages に自動push
3. スマホアプリは次回起動時に自動で最新表示

### 予約管理（家族全員）

1. アプリで「✅ 予約済」タブ
2. 「＋ 予約を追加」で日付・学校・メモ等を入力
3. **自動でGASに送信** → 家族全員に共有される
4. 他端末では「🔄 同期」を押すか、起動時に最新を自動取得

### スマホでの予約削除に注意

- PCで削除 → 自動でGASに反映
- スマホで反映するには **「🔄 同期」を押すか、起動し直す**
- 同期方針：**リモートが正、ローカル上書き**（削除も反映される）

## 地図機能

- **タイル**: 国土地理院 淡色地図（日本語・軽量・スマホ最適）
- **マーカー**: 偏差値帯で色分け（赤=65+/橙=58-64/青=50-57/緑=~49）
- **常時ラベル**: 偏差値＋校名がマーカー横に表示
- **タップ時ポップアップ**: 学校情報＋通学方法（🚃電車/🚶徒歩/🚲自転車）＋直線距離
- **高さ**: 50vh（最大450px）。地図外にスクロール領域を確保
- **慣性無効・Canvas描画**: スマホでカクつき軽減

## APK再ビルドが必要なタイミング

| 変更内容 | APK更新 |
|---|---|
| HTML/CSS/JS変更（地図デザイン・予約UI等） | **不要** ← 自動反映 |
| Pythonスクリプト変更（スクレイプロジック） | **不要** ← bat実行＋pushで完結 |
| Android本体動作の変更（SwipeRefresh等） | **必要** |
| オフライン用内蔵HTML更新 | 任意（多くは不要） |

通常は GitHub Pages 更新だけで完結する。

## 注意事項

- スマホでオフライン中に追加した予約は、オンライン復帰後の「🔄 同期」で消える可能性あり（リモートが正の方針）
  - 対策: オフライン追加直後は「同期」を押さない・自動で `rsvSyncUp` が走るのを待つ
- 偏差値は公式発表が無いため手動入力方式（minkou.jp ソース）
- URL変更された学校は取得失敗として表示
